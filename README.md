# InvoiceCraft

Professional invoice generation made simple.

InvoiceCraft is a modern web application that helps users create, preview, save, and download professional invoices directly from their browser. Whether you need an invoice for electronics, repair services, freelance work, consulting services, rentals, or custom business transactions, InvoiceCraft provides a fast and intuitive experience without requiring a backend.

The application is built as a frontend-only solution and can be deployed easily on GitHub Pages.

## Features

### Invoice Creation

* Create professional invoices in minutes
* Auto-generate invoice numbers
* Add customer and seller information
* Support for products and services
* Tax and discount calculations
* Multiple payment methods
* Notes and warranty sections

### Live Preview

* Real-time invoice preview while editing
* Instant updates for all invoice changes
* Print-ready layouts

### Invoice Templates

* Minimal
* Apple-inspired
* Business
* Dark

### Save and Manage Invoices

* Save invoices locally using browser storage
* Edit existing invoices
* Duplicate invoices
* No account required

### Export Options

* Download invoices as PDF
* Print invoices directly from the browser
* Professional A4 layout

### User Experience

* Responsive design for desktop, tablet, and mobile
* Light and dark mode
* Clean and modern interface
* Fast and lightweight

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/arundidauli/invoicecraft.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Deployment

InvoiceCraft is designed for static hosting platforms including:

* GitHub Pages
* Vercel
* Netlify
* Cloudflare Pages

To create a production build:

```bash
npm run build
```

Deploy the generated `dist` folder to your preferred hosting platform.
